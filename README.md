## API

[업비트 개발자 센터](https://docs.upbit.com/reference)

- 시세 종목 조회 API를 통해 코인들의 이름을 받아온다.
- 시세 Ticker API를 통해 특정 코인의 정보를 받아온다.
- 시세 호가 정보 API를 통해 특정 코인의 실시간 호가 정보를 받아온다.
